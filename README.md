Done and Dusted 
**A Professional, High-Performance Task Manager built with Flask.**

"Done and Dusted" is a sleek, dark-themed productivity tool designed for speed and clarity. It features a modern Single Page Application (SPA) feel with live searching, instant sorting, and smart urgency tracking.

Key Features
* **Smart Urgency:** Automatically flags tasks due within 3 days with a neon red glow and moves them to the top.
* **Live Search & Sort:** Instant filtering by Title, Priority, Alphabet, or Due Date without page reloads.
* **Task History:** A dedicated slide-out panel to track your completed goals, featuring a live status badge.
* **Responsive Layout:** The main content intelligently shifts to make room for the history sidebar.
* **Persistent State:** Remembers your sidebar preferences (open/closed) using Local Storage.

Tech Stack
 **Backend:** Python & Flask
 **Database:** SQLAlchemy (SQLite)
 **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid), and Vanilla JavaScript
